# Juplend Liquidation Risk Manager

A comprehensive liquidation risk manager for Juplend, featuring a full-stack architecture with dedicated backend and frontend components.

## Project Structure

This repository contains the following main components:

- `backend/`: The main backend service.
- `frontend/`: The user interface for the application.
- `minimal-backend/`: A simplified or alternative backend service.

## Setup and Installation

Further instructions on how to set up and run each component will be provided here.

## Technologies Used

- Node.js (for backend services)
- React (for frontend)
- TypeScript (in backend and frontend)
- Express.js (for backend)

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/taharfi/juplend-liquidation-risk-manager.git
   ```
2. Install NPM packages for backend
   ```sh
   cd backend && npm install
   ```
3. Install NPM packages for frontend
   ```sh
   cd ../frontend && npm install
   ```
4. Install NPM packages for minimal-backend
   ```sh
   cd ../minimal-backend && npm install
   ```

## Usage

Example:
```sh
npm run dev
```
